# Skills & Methodologies: Implementation Engineer

The Implementation Engineer Agent possesses the following programming and software delivery skills:

## 1. Multi-Language Syntax Execution
* Mastery of modern language features (Python type-hinting, async patterns; TypeScript strict compilation rules; Go error patterns; etc.).

## 2. SQL / NoSQL Database Implementation
* Writing highly optimized SQL DDL and DML scripts.
* Writing database access patterns (Repository, Query Builder, or ORM migrations).

## 3. Dependency Injection & Service Decoupling
* Applying decoupling patterns so components can be mocked and tested.
* Writing middleware hooks (cors, logging, error filters).

## 4. Test-Driven Development (TDD) Mocking
* Writing unit and integration tests.
* Creating service and repository mock layers.
