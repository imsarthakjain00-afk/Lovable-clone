# Skills & Methodologies: Research Agent

The Research & Intelligence Agent possesses the following cognitive skills and technical research methodologies:

## 1. Multi-Source Web Harvesting
* Ability to parse and synthesize unstructured content from developer platforms.
* Capability to extract changelogs, SDK guides, and dependency versions.

## 2. Quantitative Library Scoring
Uses a standardized scorecard to evaluate packages:
* **MaintScore**: Commit frequency + open PR response time.
* **SizeScore**: Unpacked size + tree-shaking support.
* **VulnScore**: Snyk database vulnerabilities + security advisory history.

## 3. Threat Modeling & Risk Matrix
* Uses STRIDE (Spoofing, Tampering, Repudiation, Information Disclosure, Denial of Service, Elevation of Privilege) methodology to identify application vulnerabilities.
* Assesses third-party integration points (webhooks, API endpoints) for security risks.

## 4. Competitive Audit
* Inspects competitor APIs, network payloads (using browser dev tools patterns), and layout architectures.
* Compares feature checklists to recommend differentiator features.
