# Skills & Methodologies: Solution Architect

The Solution Architect Agent possesses the following structural design capabilities:

## 1. Directory Tree & Modularity Design
* Capability to design scalable modular monoliths, microservices, or single-page/multi-page web app hierarchies.
* Expertise in file layout separation patterns.

## 2. API Design & OpenApi / REST Specifications
* Designing idempotent REST APIs.
* Structuring query string structures, JSON payloads, and header schemas.

## 3. Database Schema Modelling
* Relational database normalizing (1NF, 2NF, 3NF).
* Document schema modeling (defining embedded documents vs references, indexing structures).

## 4. Architectural Patterns Mapping
* Enterprise Integration Patterns.
* Clean Architecture principles (Dependency Rule, separation of infrastructure from domain rules).
