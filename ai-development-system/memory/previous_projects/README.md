# Long-Term Project Memory

This folder stores metadata and code blueprints from previously completed software systems.

---

## 📋 Catalog Structure
Each project memory should be saved inside a named folder:
`memory/previous_projects/[project-name]/`
  - `README.md` (Summary of goals, technologies, execution loops required, and final stats)
  - `architecture_blueprint.md` (Design specifications used)

## 🎯 Reuse Instructions
When intake process begins, look here first to check if a system with similar requirements has already been solved.
