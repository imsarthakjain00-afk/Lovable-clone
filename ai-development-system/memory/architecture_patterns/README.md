# Standard Architectural Blueprints

Reference architectural maps for scaling systems.

---

## 🏛️ Patterns Catalogue
1. **Clean Modular Monolith Layout**: Layers for Route/Controller -> Services -> Domain Entities -> Repositories.
2. **CQRS (Command Query Responsibility Segregation) Layout**: Mappings for read queries and state commands.
3. **Event-Driven Broker Interface**: Broker publish/subscribe layouts (RabbitMQ / Kafka).
