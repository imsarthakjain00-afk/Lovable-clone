# Project Risk Registry

A review of project risk vectors.

---

## 🎯 Risk Registry

* **Risk 1: Third-party API pricing changes**
  - Severity: High
  - Mitigation: Abstract integrations so switching providers requires minimal changes.
* **Risk 2: Library obsolescence**
  - Severity: Medium
  - Mitigation: Rely only on highly active open-source dependencies.
* **Risk 3: Integration complexity**
  - Severity: Medium
  - Mitigation: Validate early with isolated integration scripts.
